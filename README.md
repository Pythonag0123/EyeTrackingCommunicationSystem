# Eye Tracking Communication System (ETCS-MIU)

## Problem
Motor impaired users cannot use keyboards or mouse.

## Solution
A browser-based eye tracking communication system using MERN + MediaPipe.

## Tech Stack
- React
- Node.js
- Express
- MongoDB
- MediaPipe
- TensorFlow.js
